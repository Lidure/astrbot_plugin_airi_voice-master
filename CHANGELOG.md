## [2.2] - 2026-03-14

### ✨ 新增功能

- **🤖 LLM 触发模式（可选）**  
  - 在插件配置中为 `trigger_mode` 新增 `llm` 选项。  
  - 仅当触发模式为 `llm` 时，才会向 AstrBot 注册 `airi_list_all_voices` / `airi_search_voices` / `airi_send_voice` 等 LLM 工具。  
  - 未开启 `llm` 模式时，插件行为与旧版完全一致，只通过关键词触发语音，不参与 LLM 工具调用。

- **🧠 LLM 语音选择工具**  
  - 新增语音工具：  
    - `airi_list_all_voices`：列出当前可用语音名称，供大模型选择。  
    - `airi_search_voices`：按关键词模糊搜索语音名称（可通过 `llm_select_mode` 配置启用）。  
    - `airi_send_voice`：根据指定名称向当前会话发送对应语音。  
  - 工具返回结果主要供大模型内部使用，对用户尽量不增加额外文本噪音。

- **🎲 随机语音指令**  
  - 支持在聊天中直接发送：  
    - `随机发条语音` / `随机语音`：全局随机发送一条语音。  
    - `随机XX`：在包含 `XX` 的语音中随机挑选一条发送。

## [2.0] - 2026-03-13

### ✨ 新增功能

- **🆕 引用添加语音**  
  新增 `/voice.add 名字` 命令，支持在聊天中引用语音消息快速添加新语音
  - 无需进入后台，聊天中即可完成语音添加
  - 自动下载并保存到持久化数据目录
  - 添加后立即可用，无需重启或手动刷新

- **🗑️ 删除语音**  
  新增 `/voice.delete 名字` 命令，支持删除通过 `/voice.add` 添加的语音
  - 保护本地 `voices/` 目录文件不被误删
  - 自动同步更新配置池

- **⚙️ 配置自动持久化**  
  通过 `/voice.add` 添加的语音会自动加入 `extra_voice_pool` 配置
  - 机器人重启后语音列表依然保留
  - 无需手动编辑配置文件

### 🔒 安全改进

- 所有管理命令 (`voice.add` / `voice.delete` / `voice.reload`) 均需要管理员权限
- 路径安全检查，防止目录穿越攻击
- 只允许删除 `extra_voices/` 目录下的文件

### 📁 存储优化

- 网页上传和 `/voice.add` 添加的文件统一存储至数据目录
- 确保插件更新时用户添加的语音不会丢失

## [1.5] - 2026-3-11

### ✨ 新增功能

- **🔐 权限控制系统**  
  新增管理员权限验证，支持三种模式：
